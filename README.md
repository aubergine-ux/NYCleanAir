## Features
* **Live US AQI** - Real-time Air Quality index for NYC.
* **Color-Changing Background** - The whole page shifts color as the air gets worse.
* **Plain-English Verdict** - Tells you if it's actually okay to go outside.
* **24-Hour Trend Chart** - See if the air is getting better or worse.
* **Pollutant Breakdown** - PM2.5, Ozone, CO.

## Setup
1. Clone the repo and open `index.html` in your browser
2. To use a different city, open `index.js` and change the `lat` and `lon` values at the top
3. To modify the theme, open `style.css` and adjust your hex colors

## Data
Air quality data comes from the [Open-Meteo Air Quality API](https://open-meteo.com/en/docs/air-quality-api), based on CAMS forecasts. No API key needed.
